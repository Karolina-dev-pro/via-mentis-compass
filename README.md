# Fundacja Via Mentis

Strona internetowa Fundacji Via Mentis

## Stack techniczny

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Backend: baza danych i autoryzacja (CMS administracyjny)

## Rozwój lokalny

Wymagania: Node.js + npm.

```sh
npm install
npm run dev
```

## Build produkcyjny

```sh
npm run build
```

## Testy

```sh
npm test
```
